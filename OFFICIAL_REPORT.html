1. Executive Summary

This report details the technical and creative execution of the Rizal Shrine Dapitan website. The project aimed to bridge cultural preservation with modern interface design to create an immersive digital experience for international and local visitors.

"Journey through the four-year exile of Dr. Jose Rizal—a period of reflection, innovation, and enduring legacy."

2. System Architecture

Core Engine HTML5 / CSS3 / Vanilla JavaScript
Rendering Custom Glassmorphism Engine
Animations Intersection Observer API
Engagement Interactive Trivia & Google Maps Hub

2.1 Logic Modules

Intersection Observer: High-performance detection of scroll positions to trigger entry animations (reveals and staggers).

State Management: Used in the Trivia Quiz to track user scores and dynamic feedback results.

Media Engine: Custom Modal/Lightbox implementation for high-resolution photo inspection.

3. Visual Design & Aesthetic

The design follows a "Premium Heritage" system using translucent layers and high-contrast accents.

  • Typography: Outfit for authority, Inter for long-form readability.
  • Palette: Obsidian Green (#1B3322) and Forest Accent (#4CAF50).

4. Key Functional Modules

4.1 Illustrated Grounds Map

A specialized visual guide to the 16-hectare sanctuary, integrated into the Attractions page.

4.2 Jose Rizal Trivia Quiz

An educational engagement tool consisting of state-managed multiple-choice questions about the hero's life in Dapitan.

4.3 Interactive Location Maps

Optimized Google Maps integration for Rizal Park (Dapitan City Plaza).

5. Development Performance

Testing confirmed highly responsive behavior across all mobile and desktop devices. The animation engine maintains a constant 60FPS due to the lean avoidance of external libraries like jQuery.
