# 🏛️ Rizal Shrine Dapitan City | Official Website

A premium, modern Digital Heritage platform dedicated to preserving and showcasing the four-year exile of the Philippine National Hero, **Dr. Jose Rizal**, in Talisay, Dapitan City (1892-1896).

---

## 🌟 Premium Features

### 💎 High-End UI/UX
*   **Glassmorphism Aesthetic**: Modern, semi-transparent surfaces with deep blurs (`backdrop-filter`) and vibrant green accents.
*   **Global Animation System**: Fluid scroll-based reveals, staggered grid entrances, and smooth page transitions powered by **Intersection Observer**.
*   **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

### 📸 Immersive Media
*   **Lightbox Gallery**: Full-screen, high-resolution photo viewer with elastic "zoom" animations and integrated heritage captions.
*   **Illustrated Grounds Map**: A specialized, visual guide to the 16-hectare sanctuary (Mi Retiro Rock, Casa Cuadrada, etc.).

### 🗺️ Visitor Information
*   **Interactive Location Hub**: Google Maps integration for Rizal Park (Dapitan City Plaza), specifically tuned for visibility and clarity.
*   **Digital Concierge**: A glass-styled Inquiry Form integrated with **Formspree** for direct historical and tour inquiries.

### 🧠 Interactive Engagement
*   **Rizal Trivia Quiz**: A highly interactive, 5-question trivia challenge to test visitor knowledge about Rizal's legacy in exile.

## 🚀 Quick Start

### Running Locally
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/insongtrudge04/bayot.git
    cd bayot
    ```
2.  **Install Dependencies**:
    ```bash
    npm install
    ```
3.  **Start Development Server**:
    ```bash
    npm run dev
    ```

## 🎨 Design Philosophy
The website's palette is inspired by the **lush flora** and **earthy tones** of the Dapitan sanctuary.

*   **#1B3322 (Deep Obsidian Green)**: Used for strong headings and authoritative text.
*   **#4CAF50 (Vibrant Green)**: Used for interactive accents and high-visibility elements.
*   **Glassmorphism**: Reflects the "clarity through transparency" ideal of Dr. Jose Rizal.
